Welcome to Red-Haired Pirates Documentation
============================================

A Django web application for managing the Red Hair Pirates crew from One Piece.

Features
--------

* Crew member signup and management
* Biography pages about Shanks
* Contact information
* SQLite3 database integration
* Bootstrap 5 responsive design
* Docker support
